# Lamabot

Role:
Senior Data Scientist at LinkedIn

Problem:
Employees needed a conversational assistant to help find internal information, answer questions, and submit support tickets.

My contributions:
- Intent classification using multiple approaches: rule-based systems, Azure LUIS, and OpenAI few-shot prompting.
- Designed and executed offline evaluation for intent classification using representative test datasets and accuracy measurements.
- Developed retrieval evaluation methodologies to measure whether relevant documents appeared in top search results.
- Monitored user feedback and unresolved questions to identify failure modes and improve prompts, retrieval quality, and knowledge coverage.
- Integrated enterprise search capabilities.
- Contributed to RAG-based chatbot architecture.
- Participated in migration to OpenAI-powered solutions.
- Implemented production monitoring and observability practices.

Leadership:
- Led the end-to-end delivery of a new Contract Search capability within Lamabot.
- Gathered requirements from stakeholders and translated business needs into a secure, scalable technical solution.
- Designed an authorization workflow that validated user permissions based on team affiliations before granting access to sensitive contract information.
- Coordinated integration with multiple internal data sources and partnered with cross-functional teams to align on data access, security requirements, and deployment plans.
- Drove testing, rollout, and production adoption of the feature, enabling employees to retrieve contract information directly through the chatbot using company identifiers.
